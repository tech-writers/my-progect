== Назначение компонента
Компонент «Технический проект» содержит документы технического проекта, описывающие требования, постановку задачи, автоматизируемые функции и пояснительную записку.

== Состав модулей
* Частное техническое задание (ЧТЗ) — modules/chtz
* Описание постановки задачи (ОПЗ) — modules/opz
* Описание автоматизируемых функций (ОАФ) — modules/oaf
* Пояснительная записка к техническому проекту (ПЗТП) — modules/tp-note
